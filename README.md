How to use this feature
1: Make sure you are root (using a rooted user or adding "sudo" before the commands)
2 "mv gpio-fan-setup /bin"
3: "mv fan_setup.py /bin"
4: "chmod +x /bin/gpio-fan-setup"
5: "chmod +x /bin/fan_setup.py"
6: "gpio_fan_setup"
That's it.
This tool must be ran as root.
The tool does not check for invalid values, like gpio 67 for example. Simply enter a valid values
